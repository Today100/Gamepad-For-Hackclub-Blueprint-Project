# Gamepad - For Hackclub Blueprint Project
A hackpad that can be used to play little mini games. I think it's a fun little thing to use to take a break.

**Request:**
- Soldering Iron

BOM:
- 1x XIAO RP2040
- 16x Cherry MX Switches
- 16x Through-hole 1N4148 Diodes
- 16x Blank DSA Keycaps
- 1x SK6812 MINI LEDs
- 1x 0.91 inch OLED displays
- 4x M3x16 Bolt
